# Inspiration
I've been lurking in the crypto community for quite a while, and I've recently begun diving into the world of NFTs. Unfortunately I'm too poor to invest a couple hundred/thousand dollars into a JPEG, so I was looking for another way to participate.

I'm particularly interested in the concept of generative art — art that is generated by code — and decided to try my hand at a basic generative art project.

I had some experience with [D3.js](https://d3js.org) and [SVGs](https://developer.mozilla.org/en-US/docs/Web/SVG), so I figured I could start there.

There's a project live on Artblocks, a generative art / NFT platform, right now called [Chroma Theory](https://www.artblocks.io/project/170) that caught my eye in particular, so I decided to model my efforts after that project.

# Result
Here's my result below. You can click regenerate endlessly and each time it will generate a completely unique piece of abstract art, ultimately generated based on the exact timestamp you hit the regenerate button.

# Code

And here's the code:

```
import React, { useState } from 'react'
import { Button, Box } from 'grommet'; 
import { DateTime } from 'luxon';

var sha512 = require('js-sha512');

function ReactChroma() {

  const regenerate = () => {
      return sha512(DateTime.now().toISO()).replace(/\D/g,'')
  }
  const initial = regenerate()
  const [seed, setSeed] = useState(initial)

  let array = []
  for (let index = 0; index < seed.length; index++) {
      array.push([
          seed.charAt(index), 
          seed.charAt(index+1), 
          seed.charAt(index+3), 
          seed.charAt(index+4)
      ])
  }
  
  return (
    <Box width="500px" gap="small">
      <svg height="500px" width="500px">
        <g style={{filter: "blur(50px)"}}>
        {
          array.map((value, index) => {
            return (
              <circle 
                style={{filter: `blur(${value[3]+1*2}px)`}} 
                cx={value[0]*50} 
                cy={value[1]*50} 
                r={value[2]*15} 
                key={index} 
                fill={`rgb(${value[0]*25},${value[1]*25},${value[2]*25})`}
              >
                  <animate 
                    attributeName="r" 
                    values={`${value[2]*15};${value[0]*15};${value[2]*15}`} 
                    dur={`${value[3]*5}s`} 
                    repeatCount="indefinite" 
                  />
                  <animate 
                    attributeName="cx" 
                    values={`${value[0]*50};${value[3]*50};${value[0]*50}`} 
                    dur={`${value[0]*5}s`} 
                    repeatCount="indefinite" 
                  />
                  <animate 
                    attributeName="cy" 
                    values={`${value[1]*50};${value[2]*50};${value[1]*50}`} 
                    dur={`${value[1]*5}s`} 
                    repeatCount="indefinite" 
                  />
              </circle>
            )
          })
        }
        </g>
      </svg>
      <Button 
        label="Regenerate" 
        onClick={() => {setSeed(regenerate())}}
      />

    </Box>
  )
}

export default ReactChroma;

```
